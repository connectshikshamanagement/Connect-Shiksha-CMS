================================================================================
  WHAT IF SOMETHING GOES WRONG?
================================================================================

PROBLEM: Can't push to GitHub
‚Üí Check you created Personal Access Token
‚Üí Use token as password, not your GitHub password

PROBLEM: MongoDB won't connect
‚Üí Check you allowed 0.0.0.0/0 in network access
‚Üí Check password is correct in connection string

PROBLEM: Render build fails
‚Üí Check logs in Render dashboard
‚Üí Verify all environment variables are added

PROBLEM: Vercel shows 404
‚Üí Check Root Directory is set to "client"
‚Üí Redeploy: Vercel ‚Üí Settings ‚Üí Redeploy

PROBLEM: Frontend can't connect to backend
‚Üí Check NEXT_PUBLIC_API_URL has /api at end
‚Üí Check backend URL is correct
‚Üí Redeploy Vercel after fixing

PROBLEM: Can't login after deployment
‚Üí Run seed script on Render Shell
‚Üí Check MongoDB is connected (see Render logs)
